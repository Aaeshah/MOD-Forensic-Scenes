# MOD-Forensic-Scenes
Beyond First Glance (BFG) is a forensics-focused 3D benchmark comparing metrically scaled LiDAR and 3D Gaussian Splatting across five simulated manner-of-death scenes. Includes point clouds/meshes, pretrained 3DGS, photos/videos, ROI annotations, and a reproducible suite for geometry, coverage, efficiency, and robustness. With baselines and metrics
